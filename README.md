# Next.js App Router Course - Dashboard Starter

Welcome! This is the official starter template for the **Next.js App Router Course**. This repository contains the foundational code for the dashboard application you'll be building throughout the curriculum.

### 🚀 Project Overview

The goal of this course is to teach you the core features of Next.js by building a fully functional, professional-grade financial dashboard. 

* **Live Demo:** [View the deployed project](https://nextjs-dashboard-sigma-amber-sz66lujsew.vercel.app/)
* **Official Course:** [Next.js Learn Curriculum](https://nextjs.org/learn)

---

### 🛠️ Key Features You'll Implement

By following the course, you will learn and integrate:

* **Styling:** Tailwind CSS and CSS Modules.
* **Optimizations:** Fonts, images, and metadata optimization.
* **Routing:** File-system routing, Layouts, and Nested Routes.
* **Data Fetching:** Setting up a Vercel database and using Server Components.
* **Search and Pagination:** Implementing server-side logic using URL params.
* **Mutating Data:** Using **React Server Actions**.
* **Authentication:** Securing your dashboard with **NextAuth.js**.

---

### 🏁 Getting Started

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/vercel/next-learn-starter.git](https://github.com/vercel/next-learn-starter.git)
    cd nextjs-dashboard
    ```

2.  **Install dependencies using pnpm:**
    ```bash
    pnpm install
    ```

3.  **Run the development server:**
    ```bash
    pnpm dev
    ```

4.  **Open your browser:**
    Navigate to `http://localhost:3000` to see the result.

### 📂 Folder Structure

| Folder / File | Description |
| :--- | :--- |
| `/app` | Contains all the routes, components, and logic. |
| `/app/lib` | Reusable utility functions and data fetching logic. |
| `/app/ui` | UI components like cards, tables, and sidebars. |
| `/public` | Static assets like images and icons. |
| `/scripts` | Seeding scripts to populate your database. |
